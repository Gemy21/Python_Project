# نظام الحسابات - خلفاء الحاج

برنامج حسابات متكامل لشركة الخضروات والفواكه مبني بلغة Python مع واجهة رسومية.

## المتطلبات

- Python 3.7 أو أحدث
- مكتبة Pillow لمعالجة الصور

## التثبيت

1. تثبيت المكتبات المطلوبة:
```bash
pip install -r requirements.txt
```

## التشغيل

قم بتشغيل البرنامج باستخدام:
```bash
python main.py
```

## المميزات

- ✅ واجهة Start Menu مع الصورة كخلفية
- ✅ قاعدة بيانات SQLite لتخزين البيانات محلياً (يعمل بدون إنترنت)
- ✅ إدارة المبيعات والمشتريات
- ✅ إدارة العملاء والموردين
- ✅ إدارة المنتجات (الخضروات والفواكه)
- ✅ التقارير والخزينة

## هيكل المشروع

- `main.py` - الملف الرئيسي للتطبيق وواجهة Start Menu
- `database.py` - إعداد وإدارة قاعدة البيانات SQLite
- `requirements.txt` - المكتبات المطلوبة

## قاعدة البيانات

يتم إنشاء قاعدة البيانات تلقائياً عند أول تشغيل للبرنامج باسم `company_accounts.db`

الجداول المتوفرة:
- العملاء (customers)
- الموردين (suppliers)
- المنتجات (products)
- المبيعات (sales)
- تفاصيل المبيعات (sale_items)
- المشتريات (purchases)
- تفاصيل المشتريات (purchase_items)
- المدفوعات (payments)
